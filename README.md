# Usage:
	
	1. npm install
	
	1. ln -s webpack-serve ./node_modules/.bin/webpack-serve
	
	1. npm run dev:dll
	
	1. npm run dev:serve
	
	1. npm run build


# Features:
* DLL
* Splitchunk
* Koa router
* HTTP proxy middleware 
* Babel
* LESS
* Production build configuration for cache optimization
* Webpack serve server for devevlopment and HMR
* Mutiple output
