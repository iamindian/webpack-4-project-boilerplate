#Usage:
	
	npm install
	
	ln -s webpack-serve ./node_modules/.bin/webpack-serve
	
	npm run dev:dll
	
	npm run dev:serve
	
	npm run build


#Features:
* DLL
* Splitchunk
* Koa router
* HTTP proxy middleware 
* Babel
* LESS
* Production build configuration for cache optimization
* Webpack serve server for devevlopment and HMR
